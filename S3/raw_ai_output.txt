Gap ID: G1
Classification: G4 Decision Voids
Description: The validity duration of a waitlist seat offer is unresolved, leaving the waitlist promotion process incomplete.
Evidence:

T4: “When a seat becomes available, the first eligible student on the waitlist should be offered the seat, but we still need to decide how long that offer remains valid before it moves to the next student.”
R4: “The system shall assign an available seat to a waitlisted student when a seat becomes available.”
AC4: “When a seat becomes available, the first eligible waitlisted student is offered the seat.”
Impacted Artifacts: T4, R4, AC4
Severity: High

Gap ID: G2
Classification: G3 Inconsistencies
Description: Duplicate registration rule conflicts between requirements and acceptance criteria regarding the applicable time scope.
Evidence:

T7: “Students should not be able to register for the same course twice in the same term.”
R6: “The system shall prevent duplicate registration for the same course in the same academic year.”
AC6: “A student cannot register twice for the same course in the same term.”
Impacted Artifacts: T7, R6, AC6
Severity: High

Gap ID: G3
